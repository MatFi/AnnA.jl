AnnA.IVProblem — Method
IVProblem(
    parm::Parameters,
    range::AbstractArray,
    rate::Unitful.AbstractQuantity;
    double_sweep = true,
    alg_control = AlgControl(dtmin = 1e-20,
        dt = 1e-6,
        reltol = 1e-4,
        abstol = 1e-12,
        tend = (maximum(range) - minimum(range)) / abs(rate)
    ),
)

Creates an IVProblem. The voltage parameter V defined in parm gets overwritten by the linear voltage sweep defined as V = t-> first(range) + rate*t. AlgControl.tend is forced to be (maximum(range) - minimum(range)) / abs(rate), an overwrite can be done on the finalized object using Setfield: p = Setfield.setproperties(p::IVProblem, alg_control=AlgControl(...))
